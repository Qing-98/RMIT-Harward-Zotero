# RMIT Harvard (Zotero) – Modified CSL

This repository contains a **modified CSL (Citation Style Language) file** based on the **official RMIT Harvard Zotero style**, with minor adjustments to better match common RMIT coursework referencing outputs.

- Original (official) RMIT Harvard CSL:  
  https://www.zotero.org/styles/rmit-university-harvard

⚠️ !!! This is **not an official RMIT release**. Always verify against RMIT Easy Cite and unit-specific guidelines.

---

## Purpose

The goal of this modification is to:
- Keep consistency with **RMIT Harvard** rules
- Improve Zotero-generated references in practical assignment use
- Allow transparent version control of CSL changes

---

## Usage (Zotero)

1. Download the modified `.csl` file from this repository
2. Open **Zotero** → Preferences → Cite → Styles
3. Click **+** and add the CSL file
4. Select the style when inserting citations in Word / Google Docs

---

## Notes

- Harvard styles vary between institutions; this CSL is intended **specifically for RMIT use**
- Final responsibility for citation correctness remains with the user
